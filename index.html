About This Project

This Jeopardy game was created using the Dimension template from HTML5 UP. All animations, colors, and styling have been preserved from the original template.

The game features a fully functional Jeopardy board with categories, questions, and a scoring system. The bottom navigation allows easy access to all sections of the site.

To play, simply click on a question value, type your answer, and see if you're correct!
